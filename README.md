# Sistem Informasi Jurnal Trading & Edukasi Keuangan

Aplikasi ini adalah sistem manajemen portofolio trading berbasis web yang dibangun menggunakan **PHP Native** (Tanpa Framework). Aplikasi ini dirancang untuk mensimulasikan lingkungan kerja trader profesional dan sebagai alat edukasi literasi keuangan.

- **Edukasi Keuangan:** Melatih kedisiplinan dalam mencatat arus kas (cashflow), manajemen risiko (Stop Loss/Take Profit), dan evaluasi kinerja investasi.
- **Simulasi Profesional:** Membiasakan pengguna dengan standar pelaporan portofolio yang rapi dan terstruktur, skill yang sangat dibutuhkan di industri keuangan dan perbankan.
